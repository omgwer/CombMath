Алгоритм Куна используется для поиска максимального паросочетания в двудольном графе. Для решения этой задачи, вам необходимо представить граф в виде двудольного графа, где вершины разделены на две доли.

В данном случае, граф имеет 3 узла и выглядит следующим образом:

0 1 0
1 0 1
0 1 0
Давайте представим этот граф в виде двудольного графа. Первая доля будет содержать вершины {A, B, C}, а вторая доля будет содержать вершины {1, 2, 3}. Ребра будут соединять только вершины из разных долей.

Copy code
A --- 1
B --- 2
C --- 3
Теперь, чтобы найти максимальное паросочетание, воспользуемся алгоритмом Куна. Начнем с пустого паросочетания и будем искать увеличивающие пути, пока они существуют.

Начальное пустое паросочетание: {}

Найдем увеличивающий путь:

Возьмем вершину A.
Проверим, доступны ли для нее непомеченные соседние вершины.
В данном случае, у вершины A есть только одна непомеченная соседняя вершина 1.
Продолжим поиск рекурсивно от вершины 1.
Обновим паросочетание: {(A, 1)}.
Найдем следующий увеличивающий путь:

Возьмем вершину B.
Проверим, доступны ли для нее непомеченные соседние вершины.
В данном случае, у вершины B есть две непомеченные соседние вершины 2 и 3.
Продолжим поиск рекурсивно от вершины 2.
Обновим паросочетание: {(A, 1), (B, 2)}.
Найдем следующий увеличивающий путь:

Возьмем вершину C.
Проверим, доступны ли для нее непомеченные соседние вершины.
В данном случае, у вершины C есть только одна непомеченная соседняя вершина 3.
Продолжим поиск рекурсивно от вершины 3.
Обновим паросочетание: {(A, 1), (B, 2), (C, 3)}.
Больше увеличивающих путей не найдено, и мы получили максимальное паросочетание: {(A, 1), (B, 2), (C, 3)}.
